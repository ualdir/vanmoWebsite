# VANMO — versão estática (HTML + JS)

Esta pasta é uma versão **estática** do projeto (sem React/Vite), pronta para publicar no **GitHub Pages**.

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub e envie estes arquivos (`index.html`, `styles.css`, `app.js`, `assets/`).
2. No GitHub: **Settings → Pages**
3. Em **Build and deployment**:
   - **Source**: *Deploy from a branch*
   - **Branch**: `main` (ou `master`) e pasta `/ (root)`
4. Salve. O GitHub vai gerar a URL do seu site.

## Estrutura

- `index.html`
- `styles.css`
- `app.js`
- `assets/favicon.ico`

## Observação

- Ícones usam o CDN do **Lucide**.
- O carrossel, typewriter, scroll-reveal e contadores rodam em **JavaScript puro**.
